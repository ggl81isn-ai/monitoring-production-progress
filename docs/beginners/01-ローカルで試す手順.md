# すぐやるとよいこと（自分の PC だけ）

ここまで **新しいアカウントは不要** です（Node.js が入っていることが前提）。

## 0. 事前準備

- このフォルダ（リポジトリ）を PC に置く  
- [Node.js](https://nodejs.org/) をインストール済みにする  

## 1. レポートの見た目用 CSS をビルド（`report-preview`）

PowerShell の例:

```powershell
cd リポジトリのパス\report-preview
npm install
npm run build:css
```

**何が起きるか:** `tailwind.css` など、HTML が参照するスタイルファイルが生成されます。

## 2. 週次ダッシュボードをサンプルデータで作る（`weekly-dashboard`）

```powershell
cd ..\weekly-dashboard
npm install
npx playwright install chromium
npm run build:weekly:sample
```

**何が起きるか:**

- 同梱の **サンプル JSON** から `dist/index.html` とスタイルが作られる  
- その画面の **PNG（`dist/report.png`）** が自動で保存される  

## 3. 確認するファイル

エクスプローラーで開く:

- `weekly-dashboard\dist\index.html`（ダブルクリックでブラウザ表示）  
- `weekly-dashboard\dist\report.png`（画像ビューアで表示）  

**ここまでできれば「週次ダッシュボードの生成の流れ」は一通り把握できた状態**です。

## 補足（README にある別ルート）

README では `sample-report.json` を `notion-output.json` にコピーしてから `generate` と `screenshot` を分けて実行する手順も載っています。  
`npm run build:weekly:sample` は、そのうち「サンプルで generate → screenshot」までを一度にやるショートカットです。
